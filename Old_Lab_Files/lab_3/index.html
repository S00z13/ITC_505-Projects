How to Serve a Custom Font to a Web Page

When a browser requests a web page, it also pulls in resources like images, stylesheets, and fonts. If the font used isn’t installed on a user’s computer, we can provide it by linking to a web-hosted font (like from Google Fonts) or by self-hosting the font files.

I chose to use Google Fonts because it’s simple, fast, and reliable. I picked the Orbitron font, linked it in the <head> with a <link> tag, and then used CSS to apply it to the entire body of the document.

Addendum

I structured the page using two <article> tags — one for the explanation and one for this narrative. I used headings (<h1> and <h2>) and paragraphs for clear organization. Orbitron was used intentionally because it stands out and is definitely not a default font on most machines. I added the font using the Google Fonts CDN to make it easy to serve without hosting the file myself.
