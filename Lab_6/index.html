Addendum: I created a page with a Holy Grail layout using CSS Grid. In the main content area, I used JavaScript to create a greeting interaction that displays a personalized message when the user types their name and clicks a button. I faced a challenge with connecting the button to the JavaScript function at first, but solved it by verifying that the script was correctly linked and that my function name matched the onclick event. This project helped reinforce how HTML, CSS, and JavaScript work together.
