Addendum: I created a page using the Holy Grail layout with CSS Grid. In the main content area, I implemented a fun JavaScript interaction that displays a personalized greeting when the user types their name and clicks a button. I initially struggled with linking the script correctly, which I resolved by verifying the function name and updated tag. I also had some difficulty with spacing especially in centering the content area and properly positioning the ad label. These issues helped me better understand CSS Flexbox and alignment properties. This project reinforced how HTML, CSS, and JavaScript all work together to create responsive, interactive designs.
